## Usage

```bash
$ yarn add uptimerobot-apiv2
```

``` javascript
import UptimRobot from 'uptimerobot-apiv2';

const uptimerobot = new UptimRobot("Your API key");

uptimerobot.getAccountDetails().then(data => {
  console.log(data);
});
```
